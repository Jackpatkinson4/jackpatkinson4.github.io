Projects

Dive

Dive is a 2D mechanics-driven collectathon platformer inspired by similar games such as Celeste and developed in Unity. You play as a cave diver exploring a mysterious cave, collecting fireflies along the way. The game also has a lot of speedrunning mechanics, allowing for lots of replayability. The game was developed by the Georgia Tech game development club, VGDev, and as a member of the club I helped contribute to the game by aiding in the level design.

Itch.io Page

Thrall

Thrall is a 2D exploration based platformer developed in Unity. You play as an angel explporing a delapidated world, fighting enemies and bosses, and overcoming platforming challenges as they proceed. The game has a non-linear progressiom, focusing more on organic exploration without obvious hints as to where to go. This game was another project developed by VGDev, and I helped contribute to the game by aiding in programming the game mechanics and UI systems.

More info about the game on VGDev's website

Gun with Legs

Gun with Legs is a 2D action platformer where you play as, like the title says, a gun with legs. It involves the player grabbing ammo in order to fire against enemies, and you can also switch between various types of guns for different types of enemies. The game was developed in Unity by VGDev, and I helped contribute to the game by designing some of the guns using a pixel art software called Aseprite.

More info about the game on VGDev's website
